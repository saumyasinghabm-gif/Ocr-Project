# OCR Project

This project extracts text and structured data from images using Optical Character Recognition (OCR).

## 📌 Features
- Extract text from invoice images
- Parse structured data (items, quantity, amount)
- Save output in JSON format
- Supports image processing using Python

## 🛠️ Technologies Used
- Python
- OpenCV
- Pytesseract / OCR engine
- JSON
